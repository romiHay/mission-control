===== MISSION CONTROL: OFFLINE DOWNLOAD REQUIREMENTS =====
Below are the exact direct links to download everything required to make your web app run offline.

1. FONTS (Heebo)
------------------
Download Link: https://gwfh.mranftl.com/fonts/heebo?subsets=hebrew,latin
Instructions: Download the zip file provided by that site, extract the .woff2 files, and place them in a folder called "public/fonts" in your project.

2. LEAFLET JS & CSS (Map Library)
------------------
Download Link: https://leafletjs.com/download.html
Instructions: Download the stable .zip file (e.g., Leaflet 1.9.4). Extract it. It contains leaflet.js, leaflet.css, and an "images" folder containing the marker pins. Move them into your project.

3. TAILWIND CSS
------------------
Download Link: https://github.com/tailwindlabs/tailwindcss/releases/latest
Instructions: Download the Standalone CLI executable for Windows (tailwindcss-windows-x64.exe). 

4. MAP TILES (The Map Background)
------------------
Download Link: https://data.maptiler.com/downloads/tileset/osm/asia/israel/
Instructions: Download the .mbtiles file for the region of Israel. This will replace "cartocdn.com" as your map source.

5. OFFLINE TILE SERVER
------------------
Download Link: https://github.com/maptiler/tileserver-gl/releases
Instructions: You need this to read the .mbtiles file and serve the map data locally to your application.

6. NODE.JS (Backend & Frontend Runner)
------------------
Download Link: https://nodejs.org/en/download/
Instructions: Download the Windows installer. This is required to run both the frontend and the Express.js backend.

7. POSTGRESQL (Database)
------------------
Download Link: https://www.postgresql.org/download/windows/
Instructions: Download the interactive Windows installer.

8. NPM LIBRARIES (React, Express, etc.)
------------------
Instructions: Since you have your project folder open, run `npm install` while you STILL have internet access. This will download all Node libraries (React, Express, Postgres connector) into a folder called `node_modules`. 
You ONLY need to copy your entire project folder (INCLUDING the `node_modules` folder) to your USB stick. You do not need to download them individually.
